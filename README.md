# 🔐 AI Phishing Detection Tool

A Python-based GUI application that detects phishing attempts in emails, SMS, and URLs using rule-based and AI-based analysis.

## 🚀 Features
- Keyword-based phishing detection
- AI-based suspicious word detection
- URL analysis (IP detection, domain check)
- VirusTotal integration (optional)
- Risk level classification (Low / Medium / High)
- Export report functionality
- User-friendly GUI using Tkinter

## 🛠 Tech Stack
- Python
- Tkinter
- Regex
- Requests
- tldextract
